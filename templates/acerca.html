Sistema de Riego Automatizado

GuateRiegos 2.0

Acerca de la Aplicación

Este sistema permite optimizar el riego y aplicación de fertilizante en invernaderos utilizando drones automatizados. Desarrollado como parte del Proyecto 2 del curso de Introducción a la Programación y Computación 2.

Características Técnicas
  • Lenguaje: Python 3.x
  • Framework Web: Flask
  • Estructuras de Datos: TDAs propios (Listas, Colas, Pilas)
  • Procesamiento XML: xml.etree.ElementTree
  • Visualización: Graphviz
  • Frontend: Bootstrap 5, HTML5, CSS3, JavaScript
